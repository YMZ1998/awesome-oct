# awesome-oct

A curated list of open-source projects for Optical Coherence Tomography (OCT), covering **data parsing, annotation, deep learning segmentation, format conversion, and visualization**.

---

# Project List

|   Category    |                                                                         Project                                                                         |                             Description                             |
| ------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| Segmentation  | OCTDL <br> [https://github.com/MikhailKulyabin/OCTDL](https://github.com/MikhailKulyabin/OCTDL)                                                         | Deep learning framework for OCT segmentation (training + inference) |
| Segmentation  | keras-UNET-OCT <br> [https://github.com/SanderWooning/keras-UNET-OCT](https://github.com/SanderWooning/keras-UNET-OCT)                                  | Lightweight UNet implementation in Keras                            |
| Segmentation  | OCT-Retinal-Layer-Segmenter <br> [https://github.com/beasygo1ng/OCT-Retinal-Layer-Segmenter](https://github.com/beasygo1ng/OCT-Retinal-Layer-Segmenter) | Retinal layer segmentation (ILM, RPE, etc.)                         |
| Segmentation  | eyeseg <br> [https://github.com/MedVisBonn/eyeseg](https://github.com/MedVisBonn/eyeseg)                                                                | Advanced retinal layer and fluid segmentation                       |
| Annotation    | OCTAnnotate <br> [https://github.com/krzyk87/OCTAnnotate](https://github.com/krzyk87/OCTAnnotate)                                                       | OCT image annotation tool                                           |
| Visualization | eyelab <br> [https://github.com/MedVisBonn/eyelab](https://github.com/MedVisBonn/eyelab)                                                                | Interactive visualization and labeling UI                           |
| Data Parsing  | eyepy <br> [https://github.com/MedVisBonn/eyepy](https://github.com/MedVisBonn/eyepy)                                                                   | Python library for Heidelberg `.e2e` data                           |
| Data Parsing  | heyexReader <br> [https://github.com/ayl/heyexReader](https://github.com/ayl/heyexReader)                                                               | Low-level Heyex OCT reader                                          |
| Data Parsing  | LibE2E <br> [https://github.com/neurodial/LibE2E](https://github.com/neurodial/LibE2E)                                                                  | C++ E2E parser for production systems                               |
| Conversion    | OCT-Converter <br> [https://github.com/marksgraham/OCT-Converter](https://github.com/marksgraham/OCT-Converter)                                         | Convert OCT data to images / numpy / DICOM                          |
| Conversion    | OCTA_DICOM2IMARIS <br> [https://github.com/Henk-D/OCTA_DICOM2IMARIS](https://github.com/Henk-D/OCTA_DICOM2IMARIS)                                       | Convert OCTA DICOM to Imaris format                                 |

# Summary

This collection covers the full OCT pipeline:

* Data: parsing & conversion
* Models: segmentation
* Tools: annotation & visualization
